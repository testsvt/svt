# Замена китайских названий на русские

## Проблема

Оригинальные китайские файлы в папке `Character.edf_main` были повреждены или удалены. Когда мы попытались их восстановить, мы случайно заменили их русскими файлами из папки `Character.edf_ru`.

## Текущая ситуация

- В папке `Character.edf_ru` находятся русские файлы ✅
- В папке `Character.edf_main` теперь тоже находятся русские файлы ❌
- Оригинальные китайские файлы из `Character.edf_main` потеряны ❌

## Что нужно сделать

Для выполнения задачи замены китайских названий на русские вам нужно:

1. **Восстановить оригинальные китайские файлы** в папке `Character.edf_main`
2. **Запустить скрипт замены**, который заменит китайские названия на русские

## Варианты решения

### Вариант 1: У вас есть резервные копии
Если у вас есть оригинальные китайские файлы где-то еще:
```bash
# Скопируйте их в папку Character.edf_main
cp /path/to/original/chinese/files/*.xlsx Character.edf_main/
```

### Вариант 2: Создать китайские файлы заново
Если у вас есть доступ к оригинальным китайским данным, создайте файлы с китайскими названиями.

### Вариант 3: Использовать примеры из вашего описания
Если у вас есть конкретные примеры китайских названий, которые нужно заменить, мы можем создать тестовые файлы.

Следующие столбцы содержат названия, которые нужно заменить:
- `Grade.xlsx`: `string[32]`, `string[32].1`, `string[32].2`
- `MonsterCharacter.xlsx`: `string[64].1`
- `Class.xlsx`: `string[64].10`
- `Action.xlsx`: `string[32]`
